Excited to share that my personal portfolio is now live! 🎉

I didn’t want to create just another static portfolio website. My goal was to build something interactive, modern, and memorable—something that truly reflects my journey as a Frontend Developer and AI Enthusiast.

✨ **What makes it different?**

I integrated an AI-powered chatbot that acts as my virtual assistant. Instead of answering random questions, it is trained specifically on my profile, skills, projects, and experience. Visitors can chat with it to learn more about me in a conversational way.

🔹 **Built with:**
• Next.js & TypeScript
• Tailwind CSS
• AI Chatbot Integration
• Web3Forms for Contact Form Handling

This project helped me strengthen my skills in modern frontend development, AI integration, and creating engaging user experiences.
